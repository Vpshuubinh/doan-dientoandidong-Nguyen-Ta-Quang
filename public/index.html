Hướng dẫn sử dụng hệ thống

Thực hiện theo các bước dưới đây để tải file, chia sẻ liên kết và quản lý dữ liệu hiệu quả.

Bước 1: Đăng file

  1. Vào mục Đăng file trên thanh menu.
  2. Nhấn Chọn file hoặc kéo thả file vào vùng upload.
  3. Nhấn Đăng file để gửi file lên hệ thống.
  4. Giới hạn mỗi file: tối đa 1GB.

Bước 2: Chia sẻ liên kết

  1. Sau khi upload thành công, hệ thống sẽ tạo link tải file.
  2. Nhấn Copy link để sao chép liên kết gửi cho người nhận.
  3. Người nhận mở link và xác nhận trước khi tải xuống.

Bước 3: Quản lý file

  1. Vào Danh sách file để xem, copy link hoặc xóa file.
  2. File đã xóa tạm sẽ chuyển vào Thùng rác.
  3. Tại Thùng rác, bạn có thể xóa vĩnh viễn nếu không cần nữa.
Lưu ý: Hãy kiểm tra đúng file trước khi xóa vĩnh viễn vì thao tác này không thể hoàn tác.
